# `activate_pet`
:::infoRequires:
EcoPets
:::

:::dangerTriggered Effect
This effect requires a [Trigger](https://plugins.auxilor.io/effects/all-triggers) to activate.
:::

Activates the Pet
# Effect Syntax
```yaml
- id: activate_pet
  args:
```
